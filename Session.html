leadfoot/Session

(inner) pushCookieProperties(target, source)

HTTP cookies are transmitted as semicolon-delimited strings, with a key=value pair giving the cookies name and value, then additional information about the cookie (expiry, path, domain, etc.) as additional k-v pairs. This method takes an Array describing the parts of a cookie (target), and a hash map containing the additional information (source), and pushes the properties from the source object onto the target array as properly escaped key-value strings.
